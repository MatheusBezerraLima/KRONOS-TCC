### COMANDOS PARA USAR AO LONGO DO PROJETO:

```
git clone (link) -> Clona o repositorio para a máquina;
git pull origin main -> Atualizando para versão mais recente do projeto;
git add . -> Adiciona todas as mudanças da sua máquina para o repositório;
git commit -m "descrição do commit" -> Faz o commit (descrição) das mudanças adicionadas;
git push -u origin main -> Envia todas as alterações oficialmente para a branch, nesse caso MAIN;
```

### CRIAR UMA NOVA BRANCH A PARTIR DA MAIN:

```
git checkout main -> Verifica a branch para clonar ela;
git pull origin main -> Atualizando para versão mais recente do projeto;
git checkout -b (nome da branch) -> Cria uma nova branch. Nosso padrão feature/...;
```
